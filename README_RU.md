# Производственная гимнастика — локальное офлайн PWA

Обновлённая версия интерфейса без изменения основной логики работы.

## Что изменено в этой версии
- добавлен отдельный экран **«Инфо»** с информацией о производственной гимнастике
- непонятный плюс в шапке заменён на понятную кнопку **«Инфо»**
- добавлен красивый логотип **Газпром трансгаз Томск** в интерфейс
- на главной странице появился быстрый переход в раздел с информацией
- логика комплексов, упражнений, таймера, прогресса и рейтинга сохранена
- кнопка установки приложения перенесена в раздел **Настройки**

## Быстрый запуск на компьютере
Откройте терминал в папке проекта и выполните:

```bash
python3 -m http.server 8080
```

Затем откройте в браузере:

```text
http://localhost:8080
```

## Установка на телефон
1. Разместите содержимое папки на любом HTTPS-хостинге статических файлов.
2. Откройте ссылку на телефоне.
3. Добавьте приложение на главный экран.
4. Если раньше была установлена старая версия PWA — удалите её и установите заново.

## Структура проекта
- `index.html` — основная страница
- `app.js` — логика приложения
- `styles.css` — стили
- `sw.js` — service worker для офлайн-работы
- `assets/cards` — полные карточки комплексов
- `assets/exercises` — вырезанные изображения упражнений
- `assets/brand` — логотип
- `assets/icons` — иконки приложения

## Примечание
Все данные хранятся локально на устройстве пользователя. Общего серверного рейтинга в этой версии нет.
