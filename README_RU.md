# Производственная гимнастика — локальная офлайн-версия

Готовое PWA-приложение для телефона или компьютера.

## Что уже работает

- 4 встроенные карточки комплексов упражнений
- локальное хранение прогресса на устройстве
- серия дней, звёзды, уровень, репутация
- мягкое сгорание прогресса при пропуске
- календарь последних 30 дней
- экспорт и импорт локальных данных в JSON
- установка на главный экран как PWA
- офлайн-работа после первой загрузки

## Важно

Это **локальная версия**.

- Данные **не отправляются в интернет**
- У каждого сотрудника свой прогресс только на его устройстве
- Общего рейтинга между сотрудниками в этой версии нет

## Как запустить на компьютере

Нужен простой локальный сервер, потому что PWA и service worker не работают надёжно через `file://`.

### Вариант через Python

Откройте терминал в папке `gym_app_pwa` и выполните:

```bash
python3 -m http.server 8080
```

После этого откройте в браузере:

```text
http://localhost:8080
```

## Как установить на телефон

1. Разместите папку приложения на любом простом HTTPS-хостинге статических файлов.
2. Откройте ссылку на телефоне.
3. Добавьте приложение на главный экран.
4. После первой загрузки приложение будет работать офлайн.

## Логика начислений

- 1 завершённый комплекс в день = +10 опыта
- Серии 3, 7, 14 и 30 дней дают дополнительные звёзды
- Новый уровень героя выдаётся каждые 40 очков опыта
- При пропуске серия сбрасывается, звёзды и репутация снижаются мягко

## Файлы

- `index.html` — интерфейс приложения
- `styles.css` — оформление
- `app.js` — логика прогресса и интерфейса
- `sw.js` — офлайн-кэш
- `manifest.webmanifest` — PWA-установка
- `assets/` — карточки комплексов
- `icons/` — иконки приложения
